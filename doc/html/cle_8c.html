char* _readline ( const char *  )

Calls readline() to read input and manages the shell's command history.

Description
This function is used to call @ readline() to read input from the shell prompt, and to manage the shell's command history. It does the following:
  • Calls sfree to de-allocate the memory block of the last line buffer returned by (), if nessasary. This buffer is pointed to by line_buffer. line_buffer is aliased by pipeline_list in input_loop(). To avoid a possible double free exeption, pipeline_list should never be used to free the readline buffer.
  • Calls readline() to read input, and assigns the location of the readline buffer to line_buffer.
  • If the readline buffer is not empty, then add the line read to the history list and to the history file (HISTORY_FILE). If insertion of the new entry would result in the history file having more than MAX_NUM_ENTRIES_HISTORY_FILE, or that the history file already contains more than MAX_NUM_ENTRIES_HISTORY_FILE, then the function invokes removed_lines_from_file() to remove lines from the history file before performing the insertion.
Parameters
promt_stringThe prompt string that is displayed on the terminal.
Returns
Returns line_buffer.
See Also
sfree, line_buffer, readline(), MAX_NUM_ENTRIES_HISTORY_FILE, NUM_ENTRIES_HISTORY_FILE, remove_lines_from_file(), pipeline_list, and input_loop().

Registred function to perform command completion.

Description
This function is assigned to readline's rl_attempted_completion variable. Consequently, when the tab key is pressed on the command line, it is invoked to perform command completion on a partial text. It uses readline's rl_completion_matches function with the command_completion_generator() genarator function to find possible completions for the partial text in the list of commands in the builtins array
Parameters
partial_textThe partial text to be completed.
startThe starting position of partial_text in readline's line buffer.
endThe ending position of partial_text in readline's line buffer.
Returns
Returns an array of pointers to strings. Three outcomes are possible :
  • No possible completions found. In that case, the first element of the array points to NULL.
  • One possible completion was found. In that case, the first element of the array points to the possible completion string, and the second element points to NULL.
  • n>=2 possible completion were found. In that case, the first n elements of the array point to the n possible completion strings, and the (n + 1)th element of the array points to NULL.
See Also
readline , initialize_readline(), command_completion_genarator(), and builtins.

int initialize_history ( )

Initializes the shell's command history facilities.

Description
This function is used to initialize the shell's command history facilities. It does the following :
  • Calls using_history() to make history facilities available.
  • Sets the maximum number of enties in the shell's history list to MAX_NUM_ENTRIES_HISTORY_FILE.
  • Obtains the pathname of the shell's history file and stores the result in HISTORY_FILE.
  • Obtains the number of lines in the shell's history file (HISTORY_FILE) via count_lines_in_file() , and assigns the value to NUM_ENTRIES_HISTORY_FILE.
  • Reads history from the history file (HISTORY_FILE) into the shell's history list.
  • Disables the recording of timestamps for history entries.
Returns
Returns void.

and initialize_readline(). MAX_NUM_ENTRIES_HISTORY_FILE, HISTORY_FILE, NUM_ENTRIES_HISTORY_FILE, count_lines_in_file(),

int initialize_readline ( )

Initializes the shell's commmandline editing facilities.

Description
This function is used to initialize the shell's commandline editing facilities. It does the following :
  • Calls initialize_history() to initialize the shell's command history facilities.
  • Registers the shell's command completion function (command_completion()).
  • Registers custom binable functions (invert_case_in_region(), and toggle_editing_mode()).
  • Obtains the path of readline 's inputrc file and assign the value to INPUTRC_FILE.
  • Sets the INPUTRC environment variable.
  • Reads settings from the input file (INPUTRC_FILE).
Returns
Returns -1 on failure, or 0 on success.
See Also
initialize_history(), command_completion(), invert_case_in_region(), toggle_editing_mode(), INPUTRC_FILE, readline , inputrc, and ().
