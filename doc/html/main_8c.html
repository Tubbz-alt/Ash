void input_loop ( )

The shell's input loop.

This function runs the shell's main input loop. In an infinite loop, it does the following:

  • Calls _readline() to read input from the user.
  • Sets up yacc to read input from the readline buffer.
  • Calls yyparse() (i.e., the parser) to parse the input read.
  • Cleans up buffers
See Also
main(), _readline(), yacc, bison, yyparse(), and sfree().

The main function.

Description
This is the main function. It does the following:
  • Calls init_shell() to initialize the shell.
  • Calls initialize_readline() to initialize the shell's command line editing facilities.
  • Calls input_loop() to run the shell's input loop.
See Also
input_loop(), initialize_readline(), and init_shell().

char* pipeline_list = NULL

Points to the readline buffer.

This pointer points to the readline buffer returned by _readline(). It aliases line_buffer. To avoid a possible double free exeption, this pointer should never be used to free the readline buffer.
